# Password-Strength-Analyzer
Password Strength Analyzer
Description

This project is a Password Strength Analyzer developed using Python. It checks the strength of a password based on factors such as length, uppercase letters, lowercase letters, numbers, and special characters. The program also suggests improvements for weak passwords and generates stronger password alternatives.

To improve security, passwords are stored as SHA-256 hashes instead of plain text, and the program can detect if a password has been used before.

Features
Checks password strength
Validates password complexity
Suggests improvements for weak passwords
Generates strong passwords
Uses SHA-256 hashing
Detects password reuse
Technologies Used
Python
Regular Expressions
Hashlib
File Handling
How to Run
Download or clone the project.
Open the project folder in VS Code.
Open the terminal.
Run:
python password_strength_analyzer.py
Enter a password when prompted.
